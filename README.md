# AIGC-mobile

### 项目介绍
基于AIGC模型⽂⽣图+图⽣图的移动端及⼤屏展⽰应⽤,旨在实现⼀体化产品设计，整体分为“包装设计”“Logo设计”“海报设计”三个模块



### 运行项目

```sh
npm run dev
```
